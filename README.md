# Processing Website 2018
Static website using Gatsby & Netlify CMS

## Install dependencies
`npm install`

## Development mode
`npm run dev`

## Production build
`npm run build`

### Local Web-server
http://localhost:8000
### GraphQL REPL
http://localhost:8000/___graphql
### Netlify CMS (Login with Github credentials)
http://localhost:8000/admin/
